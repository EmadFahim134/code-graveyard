# Phone Number Tracker

A Python tool to validate and retrieve information about phone numbers, including carrier details, geographic location, and number formatting.

![Python Version](https://img.shields.io/badge/python-3.9%2B-blue)
[![License: MIT](https://img.shields.io/badge/License-BSD-yellow.svg)](https://opensource.org/licenses/MIT)

## Features

- ✅ Validate phone numbers from any country
- 🌍 Get geographic information (country, region, timezone)
- 📞 Identify carrier/service provider
- 🔢 Format numbers in international/local standards
- 🚀 Lightweight and fast (uses offline database)
- 📁 Supports both CLI and programmatic use
